5 Conclusion ET Perspectives

20

5.1. Conclusion

Objectif atteint : Création d'une application web efficace pour le dépistage de la dyslexie, avec un modèle IA performant et une API intégrée.

Collecte et traitement des données : Utilisation de 100 réponses collectées pour entraîner le modèle IA, démontrant une précision satisfaisante dans les tests.

Utilité pratique : Outil intuitif et précieux pour les orthophonistes et les parents, facilitant la gestion des tests et des résultats.

21

5.2. Perspectives

Améliorer le modèle IA : Enrichir la base de données avec plus de réponses pour augmenter la précision du modèle.

Collaborer avec les enseignants : Impliquer les enseignants pour obtenir des retours et affiner les questions du test.

Ajouter d'autres variétés de tests : Intégrer des tests supplémentaires pour diagnostiquer d'autres troubles d'apprentissage
